Who Would Want to Read this Book

This book is aimed towards users who want to use a system following the Linux From Scratch book but also want to use Steam and/or Wine. The official Linux From Scratch and Beyond Linux From Scratch books do not support these two packages. However, users should still have both the choice and an easy to follow guide to get these packages working on their system.
